Databáze

Instalace

Jako prostředí pro nasazení Informačního Systému byl zvolen školní server eva.fit.vutbr.cz

Softwarové požadavky

  • php 7.2
  • MySQL 5.6.40
  • Composer
  • doctrine orm 3.3
  • Symfony 4.3

Postup nasazení na server eva.fit.vutbr.cz

  • do adresáře WWW je potřeba vložit soubor .htaccess obsahující 
    DefaultCharset utf-8 
AddHandler application/x-httpd-php72 .php
  • Dále je třeba zkopírovat soubory aplikace do složky WWW/IIS
  • Poté je zapotřebí nastavit v souboru .env následující proměnné 
    APP_ENV=prod
DATABASE_URL=mysql://username:password@localhost/username
  • Na závěr je potřeba ve složce WWW/IIS spustit: 
    php72 composer install
php72 bin/console doctrine:migrations:migrate
php72 bin/console doctrine:fixtures:load

Kontext (basketball)

IS podporuje tři herní systémy:
  • 1x1
    • - jednotlivci
  • 3x3
    • - tři hráči na hřišti + až dva hráči na střídání
  • 5x5
    • - Pět hráčů na hřišti + až sedm hráčů na střídání
